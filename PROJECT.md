Mission

To preserve, document, digitize, and develop the Tarifit language and its dialects using artificial intelligence technologies.

Objectives

- Build the first comprehensive Tarifit dictionary.
- Create speech datasets.
- Develop translation systems.
- Build speech recognition and speech synthesis.
- Create the first AI language model for Tarifit.

Dialect Coverage

- Eastern Rif
- Central Rif
- Western Rif
- Thaqraycht
