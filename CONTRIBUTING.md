# Contributing to Tarifit-Dialects

Thank you for your interest in contributing to **Tarifit-Dialects**.

Our goal is to build a high-quality, open-source linguistic resource for the Tarifit language.

## How to Contribute

You can contribute by:

- Adding new vocabulary.
- Correcting existing entries.
- Improving translations.
- Adding example sentences.
- Documenting dialect-specific features.
- Improving documentation.

## Contribution Guidelines

- Follow the repository's orthography rules.
- Keep formatting consistent.
- Provide accurate Arabic and English translations.
- Avoid duplicate entries.
- Use the data template for new vocabulary.

## Pull Requests

Before submitting a pull request:

1. Check for duplicate entries.
2. Verify spelling and grammar.
3. Ensure all required fields are completed.

## Code of Conduct

Please be respectful and constructive when collaborating with others.

Thank you for helping preserve and develop the Tarifit language.
