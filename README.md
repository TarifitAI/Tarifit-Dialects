# Tarifit-Dialects

The official open-source repository for documenting, preserving, and developing the **Tarifit language** and its regional dialects.

This project is part of **Tarifit AI**, an initiative dedicated to creating high-quality linguistic resources for artificial intelligence, language technology, and language preservation.

---

## Dialect Groups

### Eastern Rif
- Thaqraycht (Iqrayen / Qelaya)
- Thshbḍant (Ishbḍanen)
- Thbḍarst (Imṭalsen)
- Thabuyhiyt (Ayt Buyḥyi)
- Thaytsɛidt (Ayt Saɛid)

### Central Rif
- Thwaryaghth (Ayt Waryaghar)
- Thwzint (Ayt Tuzin)
- Thimsmant (Temsman)
- Thibqwyin (Ibqoyen)
- Thijznayt (Ijznayen)

### Western Rif
- Tarifit Ghomara
- Tarifit Sanhaja Srayr
